# SmokeDictionary

这个一个个人娱乐项目，本项目意在展示市面上的各大香烟信息。并不提供购买操作，不过可以对您的购买做出借鉴作用。

本项目中香烟包含四大品牌：大陆品牌、海外品牌、港澳台品牌、历史品牌

数据来源：
香烟数据爬自：烟悦网 http://www.yanyue.cn/

爬数据用的是java 代码，具体代码这里就不展示了

用户体验后续更新中，敬请期待
